# 🎉 Festival Website (GitHub Pages Ready)

## 🚀 Setup

1. Create a new GitHub repository.
2. Upload all files from this project.
3. IMPORTANT: Open `vite.config.ts` and replace:

   base: "/your-repo-name/"

   with your actual repository name.

Example:
   base: "/festival-website/"

4. Push to the `main` branch.

## 🌍 Enable GitHub Pages

Go to:
Settings → Pages

Source: GitHub Actions

The site will automatically deploy after pushing to main.

## 🛠 Local Development

npm install
npm run dev

## 🏗 Build

npm run build
